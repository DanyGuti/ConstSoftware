Diferencia entre Internet y WorldWide Web...

Internet es la inmensa red de computadoras alrededor de todo el mundo conectadas entre sí. Por otro lado la WorldWide Web es una enorme colección de páginas que se asientan sobre la red de computadoras del internet.

Diferencias entre HTML4 y HTML5...

HTML5 es el estándar que nos permite crear páginas web en sus diferentes versiones, definiendo un lenguaje para la estructura de esta, conteniendo texto, imágenes, vídeos entre otros. Las principales diferencias entre HTML4 y HTML5 es que el HTML4 dependía de otros lenguajes para su funcionamiento, mientras que en el HTML5 no depende ni de JS, Silverlight, Java Web Start y Flash.

Elemento desaprobado para HTML5...

El que estén desaprobados los elementos para la versión de HTML5 quiere decir que ya no son utilizados, pues ya existe una mejor versión sea en CSS o en mismo HTML5. Por ejemplo:
  • font
  • dir
  • frame
Sin embargo, hay elementos que ya fueron borrados como por ejemplo:
  • rev
  • charset
  • type
  • link
  • cellpadding

Forms para HTML5...

La etiqueta de "form" es usada para recolectar información del usuario y el input. Generalmente el input es mandado a un servidor para su procesador.
